Project Title: GPU Eikonal Solver

Name: Olivier Hamel

Project Outline: Implement and evaluate an improved GPU eikonal solver based off FIM (or some other approach). Possible avenues include bucketing wavefront groups (similar in spirit to radius-stepping) to avoid redundent iterations caused by having a single expansion-front, user-defined heuristics (A*-like, likely trivial), stop-any-time incremental refinement, and non-grid inputs. Evaluation will include synthetic data-sets ('empty', random-weight/speed, regular pattern, etc), non-sythetic data-sets, and a comparison to the performance of existing GPU algorithms.
